Node-留言列表小项目

发表于 2017-05-18 | 分类于 Node.js

该项目是仿照某些网站的留言列表做的一些小功能，项目没有划分 M V C，使用的前台技术是 Bootstrap，后台使用的是 Node.js，模板引擎使用的是 EJS，数据库是 MySQL。

依赖

  • Bootstrap
  • jQuery
    • 使用 jQuery 的 ajax 做异步无刷新列表加载
  • EJS
  • Node.js
    • express
      • body-parser express 的中间件(middleware)，可以用来解析 post 请求体数据
    • mysql
    • moment
  • js-cookie
